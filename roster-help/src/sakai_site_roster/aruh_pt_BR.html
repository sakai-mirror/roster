Registro: Resumo

O que faz

A ferramenta Registro permite visualizar os nomes, fotos e perfis dos participantes do site.

Principais Conceitos

Alterar visualizações: Para ver apenas os membros de grupos específicos ou seções, use a lista suspensa "Visualizar". (Esta lista suspensa será visível apenas se os grupos ou seções existirem.)

Visualização de fotos personalizadas: Para ver as fotos personalizadas que os participantes de seu site enviou, no topo da página Registro, clique em Fotos.

Visualização de perfis: Para ver o perfil de alguém, clique no nome da pessoa.

Considerações

  • A ferramenta Registro não permite adicionar ou remover participantes a partir do seu site. Para fazer isso, você deve usar o Site Info. Veja Adicionar, editar, ou remover participantes.

  • Se você não vê a ferramenta Registro listada no menu do seu site, o proprietário do site pode ter optado por não usá-la. Os proprietários do site podem adicionar a ferramenta Registro usando o Site Info. Para obter instruções, consulte Adicionar, editar, ou remover ferramentas.

Documentação de Ajuda
