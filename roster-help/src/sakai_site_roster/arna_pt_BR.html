Usando a ferramenta Registro

Para ver seu registro, no menubar do seu site, clique em Registro.

Nota: Se você não vê a ferramenta Registro listada no menu do seu site, o proprietário do site pode ter optado por não usá-la. Os proprietários do site podem adicionar a ferramenta Registro usando o Site Info. Para obter instruções, consulte Adicionar, editar, ou remover ferramentas.

A visualização

Por padrão, os participantes são listados alfabeticamente. A contagem dos participantes aparece acima da tabela.

Para personalizar a visualização:

  • Ao lado de "Visualizar", use a lista suspensa para visualizar uma seção ou grupo de cada vez.

  • Para ordenar uma coluna diferente, clique no cabeçalho da coluna. Para inverter a ordem, clique no cabeçalho novamente.

Para ver o perfil de um participante, clique no nome da pessoa.

Imagens dos participantes

Participantes do site podem ver as imagens (ou seja, fotos ou avatares) que os outros participantes definiram em seus perfis. Para fazer isso, no topo da página Registro, clique em Fotos.

Você pode usar o menu suspenso Visualizar para escolher a visualização de todos os participantes ou apenas um grupo ou seção em particular. Você pode buscar por um participante pelo "Nome" ou "ID". Se você a permissão apropriada no site de curso, você pode escolher exibir Fotos Oficiais ou Fotos para Perfil. Você pode escolher também Ocultar Nomes.

Usando Site Info

Observação: Para completar os procedimentos deste documento, você deve possuir um papel com as permissões necessárias. Para determinar seu papel, siga as instruções em Papéis dos Participantes. Para um entendimento básico de papéis e permissões, veja Papéis e permissões: Resumo.

No site de curso ou site de projeto, quando você clicar em Site Info no menubar, você verá uma lista de participantes, se você tiver as permissões apropriadas. A lista contem os nomes e usernames e deve conter a classe, seção, ID, créditos, papel e situação do usuário.

Para editar a lista de participantes, veja Adicionar, editar ou remover participantes.
